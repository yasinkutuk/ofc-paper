Bir İktisatçıya İstatistiksel Veda: Ömer Faruk Çolak’ın Köşe Yazılarının Metin Madenciliği
This repository contains the replication code, data processing scripts, and supplementary materials for the paper:

"Bir İktisatçıya İstatistiksel Veda: Ömer Faruk Çolak’ın Köşe Yazılarının Metin Madenciliği" by Yasin Kütük.

📌 Project Overview
The ofc-paper repository is designed to ensure the reproducibility of the statistical analyses and econometric models presented in the paper. The research focuses on text mining.
